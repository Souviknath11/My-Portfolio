My Toolbox & Things I Know

HTML

HTML is Hyper Text Markup Language which is the standard markup language for documents designed to be displayed in a web browser. It is often assited by style sheet language(CSS) and Javascript(JS).

CSS

CSS is Cascading Style Sheet. It is assisted with HTML and mainly used to style the documents beautifully displayed in a web browser. It is maintained by W3C(World Wide Web Consortium).

Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, front-end web development. It contains HTML, CSS and JavaScript-based design templates for typography, forms, and other interface components.

MS Office

Microsoft Office, or simply Office, is a discontinued family of client software, server software, and services developed by Microsoft. It was first announced by Bill Gates on August 1, 1988, at COMDEX in Las Vegas.

Content Writing

Content writing is the process of planning, writing and editing web content, typically for digital marketing purposes. It can include blog posts and articles, scripts for videos and podcasts, content for specific platforms.

DaVinci Resolve

DaVinci Resolve is a color grading, color correction, visual effects, and audio post-production video editing application for macOS, Windows, and Linux, developed by Blackmagic Design.
